# Proyecto PHP

## Descripción:
Para trabajar en este proyecto de creación de un codigo y una imagen co el mensaje de Hello World o en su caso, Hola mundo, se necesitaron de varios pasos y prerrequisitos para poder lograr un trabajo adecuado y eficaz que nos brinde un proceso bien estructurado.


### Prerrequisitos 
Para trabajar en este proyecto se necesita la instalacion del lenguaje de programación PHP, que es un lenguaje de codigo abierto que nos enseña el resultado en HTML; y tambien la instalacion de nuestro sistema de control de versiones, que nos ayudara a rastrear cambios en carchivos de codigo fuente durante el desarrollo de un Software.

Para mi caso, utilice una maquina virtual, VitualBox, que tiene como sistema operativo Ubunto basado en Debian Linux, junto con nuestro servidor web HTTP de codigo abierto, que es Apache2, que lo utilizamos para alojar sitios web.
También se intalo nuestro editor de código para poder realizar el proyecto ayudandome de la terminal para poder ejecutar los enlazes.


### Pasos para la ejecución:
⦁	Abrimos el folder de Html (index.html).
⦁	Abrimos la IP de Apache 2 que seria: http://localhost/.
⦁	Utilizamos los permisos: sudo chmod 777 index.html. Que nos ayuda a poder editar la pagina. 
⦁	Ahora con el siguiente comando: cd html entramos a la carpeta de HTML.
⦁	Creamos un nuevo folder para poder trabajar en una pagina web con PHP.
⦁	Una de las hojas del folder tiene que llamarse index.php.
⦁	En la hoja que acabamos de crear hacemos un codigo que nos muestre el mensaje “Hola Mundo” junto con una Imagen.
⦁	Se completa el proyecto.
